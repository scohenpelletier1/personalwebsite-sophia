Rab-Bit

The Rab-Bit game that was created during the Lightning Round of NHSGA

About the Project

Rab-Bit was a direct inspiration of the game Ladybug. Ladybug is a game that's similar to Pac-Man, with one main difference: you have the ability to change the map with rotating doors.

In one week, our team was able to create a pixelated game (an homage to the original Ladybug), complete with a home scree, lose screen, sound effects, fully integrated art, and random enemy movement that becomes aggressive when you get close.

Features

  • Collect carrots to score points
  • Pick up powerups for various different effects, like speed boosts, shields, and more
  • Explore the four different seasons with different maps at each level
  • Avoid the various enemies, seasonally themed
